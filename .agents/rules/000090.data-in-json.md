**JavaScript Data**
 * Information that is usually loaded from a database and added to the page via a template should never be hard-coded. Rather, such data should be created as a json object and live in a json file for each use in the json directory. One file per array/use (bfg-groups.json, gallery-images.json etc.).
 * If needed, functions should be made that call the appropriate file to lookup the data that should be displayed with callback functions to render them on page.
