## Multi-File Awareness
- Check for dependencies across files: variables, tables, data sources, and resources may be referenced in the `src/` folder or subdirectories
- When modifying a resource, search all references to it, including those that may exist only in code comments
- Consider module boundaries and inter-module dependencies
- Often times, code for new projects is based on code from prior projects, so do not assume all code present is meant for the current project
